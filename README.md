# 净智盒子lite——基于Arduino的智能空气净化检测提示系统

## 项目介绍
本项目是一个基于Arduino的环境监测系统，能够实时检测空气中的PM2.5、甲烷（CH4）、一氧化碳（CO）和二氧化碳（CO2）浓度，并对空气质量状况进行显示和报警。目标是通过传感器控制硬件设备来保护人们免受空气污染的危害。

## 功能特性

- **空气质量监测**：实时检测PM2.5、CO、CH4和CO2的浓度。
- **报警功能**：当污染物浓度超过安全阈值时，进行警告和长时间报警。
- **通风控制**：能够自动启用通风设备改善空气质量。
- **LCD 显示**：实时显示传感器数值并给出空气质量提示。
- **串口输出数据**：支持通过串口输出调试信息。
- **错误检测**：内置传感器错误处理功能。

## 系统组件

### 硬件

- LiquidCrystal_I2C LCD (16x2)
- PM2.5传感器
- MQ-4甲烷传感器
- JW01二氧化碳传感器
- MQ-7一氧化碳气体传感器
- 杜邦线、电路板
- 程序控制继电器
- 模拟蜂鸣器
- Arduino开发板

### 软件和框架:
- Arduino IDE
- `LiquidCrystal_I2C` 和 `SoftwareSerial` 库
